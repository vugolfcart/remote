# remote
remote control center

## Controls
- increase speed
  - up arrow
  - "w"
- decrease speed
  - down arrow
  - "s"
- stop (set speed to 0)
  - "q"
- toggle power steering (whether the wheel rotates back to normal)
  - "p"
- turn wheel
  - click & drag
  - double click and move mouse
